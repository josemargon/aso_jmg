# PR0303: Ejercicios sobre los comandos while, until y for

## 1. Contar hasta 10 con for

Usa un bucle for para contar del 1 al 10 e imprimir cada número en una línea nueva.

```
#!/bin/bash

for i in {1..10}
do
    echo $i
done
```

## 2. Sumar los primeros 50 números

Usa for para sumar los primeros 50 números naturales y muestra el resultado.

```
#!/bin/bash

suma=0

for i in {1..50}
do
    suma=$((suma + i))
done

echo "La suma de los primeros 50 números naturales es: $suma"
```

## 3. Tabla de multiplicar

Pide un número al usuario y usa for para imprimir su tabla de multiplicar del 1 al 10.

```
#!/bin/bash

echo "Introduce un número para ver su tabla de multiplicar:"
read numero

for i in {1..10}
do
    resultado=$((numero * i))
    echo "$numero x $i = $resultado"
done
```

## 4. Imprimir cada letra

Pide una palabra al usuario y usa for para imprimir cada letra en una línea nueva.

```
#!/bin/bash

echo "Introduce una palabra:"
read palabra

for (( i=0; i<${#palabra}; i++ ))
do
    echo "${palabra:$i:1}"
done
```

## 5. Contar números pares del 1 al 20 con while:

Usa while para mostrar los números pares entre 1 y 20.

```
#!/bin/bash

numero=2

while [ $numero -le 20 ]
do
    echo $numero
    numero=$((numero + 2))
done
```

## 6. Suma de dígitos

Pide un número al usuario y usa while para sumar todos sus dígitos. Por ejemplo, 123 debería devolver 6.

```
#!/bin/bash

echo "Introduce un número:"
read numero

suma=0

while [ $numero -gt 0 ]
do
    digito=$((numero % 10))  # Obtiene el último dígito
    suma=$((suma + digito))  # Suma el dígito al total
    numero=$((numero / 10))  # Elimina el último dígito del número
done

echo "La suma de los dígitos es: $suma"
```

## 7. Cuenta regresiva

Pide al usuario un número inicial y usa until para hacer una cuenta regresiva hasta llegar a cero.

```
#!/bin/bash

echo "Introduce un número inicial para la cuenta regresiva:"
read numero

until [ $numero -le 0 ]
do
    echo $numero
    numero=$((numero - 1))
done

echo "¡Cuenta regresiva terminada!"
```

## 8. Imprimir solo archivos .txt

Usa for para iterar sobre todos los archivos en un directorio y mostrar solo aquellos que terminen en .txt.

```
#!/bin/bash

echo "Introduce el directorio:"
read directorio

for archivo in "$directorio"/*.txt
do
    if [ -f "$archivo" ]; then
        echo "$archivo"
    fi
done
```

## 9. Factorial de un número

Solicita un número al usuario y calcula su factorial usando un bucle for.

```
#!/bin/bash

echo "Introduce un número para calcular su factorial:"
read numero

# Comprobar si el número es negativo
if [ $numero -lt 0 ]; then
    echo "El factorial no está definido para números negativos."
    exit 1
fi

factorial=1

for (( i=1; i<=numero; i++ ))
do
    factorial=$((factorial * i))
done

echo "El factorial de $numero es: $factorial"
```

## 10. Verificar contraseña

Usa until para pedir al usuario que ingrese una contraseña correcta, y repite hasta que la acierte.

```
#!/bin/bash

# Contraseña predefinida
contraseña_correcta="miContraseña123"

# Solicitar la contraseña al usuario
until [ "$contraseña" == "$contraseña_correcta" ]
do
    echo "Introduce la contraseña:"
    read -s contraseña  # La opción -s oculta la entrada de la contraseña
    if [ "$contraseña" != "$contraseña_correcta" ]; then
        echo "Contraseña incorrecta, intenta de nuevo."
    fi
done

echo "Contraseña correcta, acceso concedido."
```
