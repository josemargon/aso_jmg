# PR0302: Ejercicios comando case

## Ejercicio 1: Menú de operaciones matemáticas

1. Crea un script que presente un menú con las operaciones básicas (suma, resta, multiplicación, división) y solicite al usuario que elija una operación. El script debe ejecutar la operación seleccionada utilizando case.

```
#!/bin/bash

echo "Seleccione una operación:"
echo "1) Suma"
echo "2) Resta"
echo "3) Multiplicación"
echo "4) División"

read opcion

echo "Introduce el primer número:"
read num1

echo "Introduce el segundo número:"
read num2

case $opcion in
    1)
        resultado=$((num1 + num2))
        echo "El resultado de la suma es: $resultado"
        ;;
    2)
        resultado=$((num1 - num2))
        echo "El resultado de la resta es: $resultado"
        ;;
    3)
        resultado=$((num1 * num2))
        echo "El resultado de la multiplicación es: $resultado"
        ;;
    4)
        if [ $num2 -ne 0 ]; then
            resultado=$((num1 / num2))
            echo "El resultado de la división es: $resultado"
        else
            echo "Error: No se puede dividir entre cero."
        fi
        ;;
    *)
        echo "Opción inválida. Por favor, selecciona una opción válida."
        ;;
esac
```

## Ejercicio 2: Identificación de extensión de archivo

2. Haz un script que solicite al usuario un nombre de archivo y, usando case, determine si es un archivo de texto (.txt), un archivo comprimido (.zip o .tar.gz), o cualquier otro tipo de archivo.

```
#!/bin/bash

echo "Introduce el nombre del archivo:"
read archivo

case $archivo in
    *.txt)
        echo "El archivo '$archivo' es un archivo de texto."
        ;;
    *.zip | *.tar.gz)
        echo "El archivo '$archivo' es un archivo comprimido."
        ;;
    *)
        echo "El archivo '$archivo' es de un tipo desconocido."
        ;;
esac
```

## Ejercicio 3: Conversor de unidades

3. Crea un script que ofrezca un menú para convertir entre diferentes unidades de longitud (metros a kilómetros, pies a metros, etc.) utilizando case para gestionar las conversiones.

```
#!/bin/bash

echo "Seleccione la conversión que desea realizar:"
echo "1) Metros a Kilómetros"
echo "2) Kilómetros a Metros"
echo "3) Pies a Metros"
echo "4) Metros a Pies"

read opcion

case $opcion in
    1) resultado=$((valor / 1000)) ;;
    2) resultado=$((valor * 1000)) ;;
    3) resultado=$((valor * 3048 / 10000)) ;;
    4) resultado=$((valor * 10000 / 3048)) ;;
    *) echo "Opción no válida"; exit 1 ;;
esac
```

## Ejercicio 4: Menú de configuración del sistema

4. Realiza un script que presente un menú con varias opciones de configuración del sistema: por ejemplo, apagar, reiniciar o cerrar sesión. Usa case para gestionar las opciones seleccionadas.

```
#!/bin/bash

echo "Seleccione una opción:"
echo "1) Apagar el sistema"
echo "2) Reiniciar el sistema"
echo "3) Cerrar sesión"

read opcion

case $opcion in
    1)
        echo "Apagando el sistema..."
        sudo shutdown now
        ;;
    2)
        echo "Reiniciando el sistema..."
        sudo reboot
        ;;
    3)
        echo "Cerrando sesión..."
        pkill -KILL -u $USER
        ;;
    *)
        echo "Opción inválida. Por favor, seleccione una opción válida."
        ;;
esac
```

## Ejercicio 6: Clasificación de notas

6. Realiza un script que solicite una calificación numérica y, usando case, la clasifique como “Sobresaliente”, “Notable”, “Aprobado”, o “Suspenso” según el rango de valores.

```
#!/bin/bash

echo "Introduce la calificación numérica (0-10):"
read calificacion

case $calificacion in
    9|10) echo "Sobresaliente" ;;
    7|8) echo "Notable" ;;
    5|6) echo "Aprobado" ;;
    [0-4]) echo "Suspenso" ;;
    *) echo "Calificación no válida. Por favor, introduce un número entre 0 y 10." ;;
esac
```

## Ejercicio 7: Clasificación de números

7. Haz un script que solicite un número y lo clasifique como “Positivo”, “Negativo” o “Cero”.

```
#!/bin/bash

echo "Introduce un número:"
read numero

case $numero in
    [0-9]*)
        echo "El número $numero es Positivo."
        ;;
    -[0-9]*)
        echo "El número $numero es Negativo."
        ;;
    0)
        echo "El número es Cero."
        ;;
    *)
        echo "Por favor, introduce un número válido."
        ;;
esac
```

## Ejercicio 8: Control de servicios en Linux

8. Crea un script que pregunte por el nombre de un servicio y luego presente un menú para iniciar, detener o reiniciar un servicio en Linux (como apache2 o nginx). Usa case para gestionar las opciones y los comandos correspondientes (systemctl start, stop, restart).

Después de realizar la operación solicitada comprueba su código de estado de finalización (recuerda que puedes obtener el estado de finalización de un comando con la variable $? tras ejecutarlo) y muestra un mensaje indicando si la operación se ha realizado correctamente o no.

```
#!/bin/bash

# Solicitar el nombre del servicio
echo "Introduce el nombre del servicio (por ejemplo, apache2 o nginx):"
read servicio

# Mostrar el menú de opciones
echo "Selecciona una opción para el servicio $servicio:"
echo "1) Iniciar"
echo "2) Detener"
echo "3) Reiniciar"

read opcion

# Usar case para gestionar la opción seleccionada
case $opcion in
    1) sudo systemctl start $servicio ;;
    2) sudo systemctl stop $servicio ;;
    3) sudo systemctl restart $servicio ;;
    *) echo "Opción no válida"; exit 1 ;;
esac
```
