# PR0501 Creación de dominio

## Instalación de Active Directory

Para instalar Active Directory debemos asignar primero una ip estática al equipo y asignarle un nombre.

Para instalar Active Directory seleccionamos Administrar > Agregar roles y características y seleccionamos el rol de "Servicios de dominio de Active Directory", y en características "Administración de directivas de grupo".
